# Portfolio

Inspired by [@benjitaylor](https://github.com/benjitaylor)'s [personal website](https://benji.org/), this is a minimalist portfolio built with Next.js 13, Tailwind CSS, and MDX.

Also inspired by [@evisdrenova](https://github.com/evisdrenova)'s [personal website](https://evis.dev/), this portfolio is designed to be simple, fast, and easy to maintain. It features a clean design with a focus on content.
