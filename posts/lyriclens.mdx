# Overview

LyricLens is a semester project created by 5 students at Rochester Institute of Technology for the class SWEN-514: Engineering Cloud Software Systems. Throughout the semester, we designed a full-stack application using Infrastructure as Code (IaC) principles, leveraging AWS services, and implemented it using Terraform.

Our team has learned a lot of valuable skills in cloud computing, software engineering, and project management through this project. The application allows users to input a Spotify playlist and generates an image for the playlist based on the lyrics of the songs in the playlist.

In our course, teams could either choose to build a project based off a given template or create their own project from scratch and get approval from the Teaching Assistants (TAs) and the Professor. Our team decided to create our own project from scratch. We brainstormed various ideas and eventually settled on the concept of LyricLens.

# Architecture

```python
Front End (Next.js)
+ TypeScript

Back End
+ Lambda Function (Python)
+ API Gateway
+ S3 (for storing generated images and building the front end)
+ DynamoDB (for storing generated IDs and their corresponding image URLs)
+ Bedrock (for generating images based on lyrics)
+ Comprehend (for analyzing lyrics and extracting themes and sentiments)
+ Terraform (for infrastructure as code)
```

# What I Did

I was responsible for integrating the front end with the back end and ensuring that the application was functioning properly. It really wasn't the typical Next.js CORS issue where you add an authorized domain in config; this was dealing with API Gateway through Terraform.
